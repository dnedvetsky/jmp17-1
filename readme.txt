This is a very basic application;
Gets, posts, deletes from list.
Required deploy on Tomcat.

Used Jersey 2.x, to navigate through rest:

http://localhost:8080/RESTfulExample/rest/hello/userpost
http://localhost:8080/RESTfulExample/rest/hello/username/{index}
http://localhost:8080/RESTfulExample/rest/hello/putuser/{index}/put
http://localhost:8080/RESTfulExample/rest/hello/user/{index}/delete
